# Siddharth Pundir Portfolio

Personal portfolio and blog starter for `siddharthpundir.com`.

## What is included

- Dark portfolio homepage inspired by the reference design
- Profile photo area
- One-click resume download
- Skills, projects, blog preview, and contact sections
- X, LinkedIn, GitHub, and email links
- Mobile responsive layout
- Vercel-friendly static deployment

## Update before launch

- Replace `assets/profile-photo.svg` with your real photo.
- Your final resume PDF is saved at `resume/Siddharth_Pundir_Resume_Final.pdf`.
- Project GitHub and live demo links can be added when ready.
- Replace sample projects and blog posts with your real work.

## Deploy on Vercel

Import this folder/repository into Vercel. Since it is a static site, no build command is required.
